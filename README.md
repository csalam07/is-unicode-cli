# is-unicode-cli

> Let you know whether the terminal supports Unicode or not

<br>

## Installation

### npm

```sh
npm install is-unicode-cli
```

### Yarn

```sh
yarn add is-unicode-cli
```

## Usage

```js
import isCliUnicode from 'is-unicode-cli';

isCliUnicode();
// return either true or false
```
